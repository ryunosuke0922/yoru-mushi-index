# Observation Calibration

実観測に基づいて夜虫指数を調整するための方針です。

## 目的

スコアの重みを、手元の実観測ログと照合しながら段階的に調整します。
特定地点、街灯、採集地、生息地、希少種のリアルタイム位置を扱わず、粗いエリア単位の傾向だけを利用します。

## 収集候補

- 観察結果: 多かった / 普通 / 少なかった
- 分類: 蛾 / 甲虫 / 水辺の羽虫 / その他
- 場所タイプ: 住宅地 / 林縁 / 水辺 / 山地
- 日時: 夜間の時間帯
- エリア: 公開済みの粗い areaId

## 使い方

- 予報スコアと観察結果のズレを月別、気温帯、湿度帯、風速帯で集計する。
- 20-24℃帯の上限、月別の季節係数、風速・湿度の重みを優先して見直す。
- サンプル数が少ないエリアでは個別補正せず、全国または地方単位の傾向として扱う。

## 禁止事項

- exact latitude / longitude を保存しない。
- 街灯、林道、橋、公園内の具体地点を保存しない。
- 希少種のリアルタイム出現情報を保存・公開しない。
- 観察ログから具体的な採集地を推定できる表示をしない。
