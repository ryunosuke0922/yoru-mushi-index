# Location Policy

このサービスは、具体的な観察地点、街灯、採集地、生息地を公開しません。

## 方針

- 公開表示はエリア単位に限定する
- 緯度経度を直接公開しない
- API は公開可能な粗いエリア情報だけを返す
- 希少種・人気種・局所性の高い分類群の地点情報は扱わない
- 観察地ランキングを作らない
- 「おすすめポイント」機能を作らない

## 表示してよい情報

- 市区町村程度の広さ
- 20km 程度以上の粗いエリア
- 気象条件
- 月条件
- 時間帯別の活性指数
- 分類群レベルの傾向

## 表示しない情報

- 街灯
- 林道名
- 橋名
- 公園内の具体地点
- 正確な緯度経度
- 過去の詳細観察地点
- 希少種のリアルタイム出現情報

## 実装上のルール

- UI には `publicArea()` が返す情報だけを使う
- API レスポンスに `latitude` / `longitude` を含めない
- 将来投稿機能を追加する場合も、保存前に粗いエリアへ変換する
