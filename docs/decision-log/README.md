# Decision Log

設計上の重要な判断を ADR として残します。

## Naming

```txt
NNNN-short-kebab-title.md
```

Example:

```txt
0001-nextjs-monorepo.md
```

## Status

| 状態       | 意味            |
| :--------- | :-------------- |
| Draft      | 草案            |
| Proposed   | レビュー中      |
| Accepted   | 採用            |
| Rejected   | 却下            |
| Superseded | 後続 ADR で置換 |
