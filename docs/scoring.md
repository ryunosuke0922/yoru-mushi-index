# Scoring

夜虫指数は、夜間に飛翔する昆虫の観察しやすさを 0-100 で表すルールベースの指数です。

## 入力

- 気温
- 湿度
- 降水量
- 風速
- 突風
- 雲量
- 月照度、月高度、雲量による月明かりの見え方
- 季節スコア
- 生息環境スコア
- 直近 24 時間雨量

## 総合スコア

`packages/core/src/score.ts` の `calculateNightInsectScore()` で算出します。

基本方針:

- 暖かい時間帯を加点
- 湿度が高い時間帯を加点
- 弱風を加点
- 強風、突風、観察時間中の降水を減点
- 直近 24 時間雨量があり、観察時間中に雨がない場合は加点
- 月が高く明るく、雲量が少ない場合は減点
- 季節スコアと生息環境スコアを加点

## ラベル

`packages/core/src/labels.ts` の `scoreLabel()` で表示ラベルを返します。

```txt
85-100  とても高い
70-84   高い
50-69   普通
30-49   低い
0-29    とても低い
```

## 分類別スコア

`calculateTaxaScores()` で総合スコアに分類群ごとの補正を加えます。

- 蛾: 生息環境スコアを強めに反映
- 甲虫: 総合スコアと気温の影響を反映
- 水辺の羽虫: 直近 24 時間雨量と生息環境スコアを反映

## 理由

`packages/core/src/reasons.ts` の `scoreReasons()` で、ユーザーに表示する理由文を生成します。
