# Scoring

夜虫指数は、夜間に飛翔する昆虫の観察しやすさを 0-100 で表すルールベースの指数です。

## 入力

- 気温
- 湿度
- 降水量
- 風速
- 突風
- 雲量
- 月照度、月高度、雲量による月明かりの見え方
- 月別の季節係数
- エリア別の季節スコア
- 生息環境スコア
- 代表時刻より前の直近 24 時間雨量

## 総合スコア

`packages/core/src/score.ts` の `calculateNightInsectScore()` で算出します。

初期値は `35` です。そこに気温、風、雨、湿度、月明かり、季節、広域環境を加点・減点します。

基本方針:

- 暖かい時間帯を加点
- 低温時は最終スコアに上限をかける
- 湿度が高い時間帯を加点
- 弱風を加点
- 強風、突風、観察時間中の降水を減点
- 直近 24 時間雨量があり、観察時間中に雨がない場合は加点
- 月が高く明るく、雲量が少ない場合は減点
- エリア別の季節スコアを加点
- 月別の季節係数を最終スコア全体に掛け、時期による活動量の差を反映
- 生息環境スコアを加点

## ラベル

`packages/core/src/labels.ts` の `scoreLabel()` で表示ラベルを返します。

```txt
85-100  とても高い
70-84   高い
50-69   普通
30-49   低い
0-29    とても低い
```

## 分類別スコア

`calculateTaxaScores()` で総合スコアに分類群ごとの補正を加えます。

分類別スコアは、総合スコアを土台にして分類群ごとの感度差を出すための表示用スコアです。総合スコアには既に気温、雨量、生息環境が含まれているため、分類別補正は意図的な追加補正です。

```ts
moths = totalScore + habitatScore * 8;

beetles = totalScore - 12 + temperatureScore(temperature) * 0.4;

aquaticInsects =
  totalScore - 4 + Math.min(recentRainMm24h, 12) * 1.2 + habitatScore * 4;
```

補足:

- 蛾: 生息環境スコアを総合より少し強めに反映します。総合スコア側の `habitatScore * 10` に加えて、分類別で `habitatScore * 8` を追加します。
- 甲虫: 総合スコアから `12` 点下げ、気温スコアの `40%` を戻します。現行の `temperatureScore()` では 26℃以上は `18` 点です。
- 水辺の羽虫: 総合スコアから `4` 点下げ、直近 24 時間雨量を最大 12mm まで `1mm = 1.2 点` として加点し、生息環境スコアも追加します。

例:

```txt
totalScore = 70
habitatScore = 0.8
temperatureScore(26℃) = 18
recentRainMm24h = 8

蛾:
70 + 0.8 * 8 = 76.4 → 76

甲虫:
70 - 12 + 18 * 0.4 = 65.2 → 65

水辺の羽虫:
70 - 4 + 8 * 1.2 + 0.8 * 4 = 78.8 → 79
```

この方式は実装が単純で、総合スコアとの関係が読みやすい一方、総合スコアに含まれる要素を分類別で再加点する面があります。将来的に精度を上げる場合は、総合スコアの内訳と分類別固有スコアを分ける余地があります。
